About Us

Halo , Welcome to EcoSpirit E-Bike Rentals This website is a place for you to rent an electric motorbike more easily. The spirit to save the world is our goal by presenting environmentally friendly vehicles that are very reliable.

Technologies for building this website include Express.js, Node.js, Knex, PostgreSQL, PG (node-postgres), Nodemon, Bootstrap, and Postman

This Website have CRUD API. CRUD API is an interface that provides the following fundamental operations for managing data or resources:

  • Create (POST): Allows the creation of new resources in the system.
  • Read (GET): Enables the retrieval of data, including single resources or collections.
  • Update (PUT/PATCH): Supports the modification of existing resources. PUT replaces the entire resource, while PATCH updates specific fields.
  • Delete (DELETE): Permits the removal of resources from the system.
